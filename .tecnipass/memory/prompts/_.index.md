# Prompt Memory

> Archived human prompts that start, resume, or materially redirect TecniPass work.

This sub-layer preserves the original user intent before agents summarize, decompose, or execute it.

## Contents

| File | Purpose | When to load |
|---|---|---|
| `_template.prompt.md` | Template for archiving a new prompt | Before storing a new prompt |
| `YYYY-MM-DD-{short-name}.prompt.md` | Archived original prompt and routing metadata | Similarity search, session resume, audit |

## Access Pattern

1. For a substantial human prompt, run `workflows/prompt-intake.workflow.md`.
2. Archive the prompt in this folder before task decomposition.
3. Search this folder, `sessions/`, and `tasks/` for similar prior work.
4. Decide whether to resume an existing session or create a new one.
5. Reference the prompt file from the task or session that executes the work.

## Prompt File Rules

- File name: `YYYY-MM-DD-{short-kebab-name}.prompt.md`.
- Store the original prompt verbatim unless it contains secrets.
- If secrets are present, redact the secret value and state what was redacted.
- Keep extracted metadata short; do not rewrite the original prompt.
- A prompt archive is not a task by itself. Execution state belongs in `tasks/` and `sessions/`.

## Similarity Search Keys

When checking for related work, search by:
- route or endpoint, such as `/reception/invitation`;
- query params, such as `mode=continue` or `invitationId`;
- user type, registration status, temporal status, or business state;
- visible UI labels and button text;
- affected field names, hooks, validators, or components;
- named skills requested by the human.

## Resume Decision

Resume an existing session when:
- the prompt describes the same user flow and same unresolved issue;
- an active session or snapshot exists;
- the previous work is not complete or validation was blocked.

Create a new session when:
- no matching session exists;
- prior work is completed and the new prompt changes scope;
- the prompt asks for a new implementation, new validation, or new harness capability.

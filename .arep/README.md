# AREP: ML, Deep Learning & AI Bootcamp

This directory serves as the centralized documentation hub and oversight for the **AREP** (Architecting and Programming Research) course labs, focusing on Machine Learning, Deep Learning, and Cloud-Ready AI Infrastructure.

## 📌 Project Overview
The core of this work is a 4-week intensive Machine Learning Bootcamp. The goal is to master building intelligent systems from first principles, ensuring they are scalable, performant, and ready for enterprise-grade deployment.

### Key Focus Areas:
- **ML Foundation**: Implementing algorithms from scratch (Linear/Polynomial Regression, etc.).
- **Infrastructure**: Cloud-native execution and MLOps.
- **Inteligent Systems**: Embedding AI as a first-class quality attribute in modern architecture.

## 📜 Development Rules & Standard Operating Procedures

To maintain high standards across the bootcamp, all development must adhere to the following rules:

### 1. Linguistic Standard
- **English Only**: All documentation, READMEs, code comments, and variable naming MUST be in English.
- **Professionalism**: Maintain a clear and technical tone in all written artifacts.

### 2. Technology Stack
- **Core Language**: `Python 3.11+`
- **Interfaces**: `Jupyter Notebooks` (`.ipynb`) for experimentation and documentation.
- **Cloud Infrastructure**: `AWS SageMaker` for scalable execution and cloud evidence.
- **Computation**: High performance via `NumPy` vectorization.

### 3. Methodology
- **First Principles**: Implement core logic without high-level ML libraries (scikit-learn, etc.) when specified, to ensure a deep understanding of the underlying mathematics.
- **Documentation First**: Every repository must contain a comprehensive `README.md` and evidence of cloud execution (screenshots/videos).
